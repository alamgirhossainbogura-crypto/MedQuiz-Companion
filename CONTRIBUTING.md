# Contributing to MedQuiz Companion

We love your input! Whether you are filing an issue, submitting a bug fix, or suggesting a feature, here is how you can help:

1. Fork the repository.
2. Create a new branch (`git checkout -b feature/YourFeatureName`).
3. Commit your changes with clear messages.
4. Push to the branch (`git push origin feature/YourFeatureName`).
5. Open a Pull Request.

Please ensure your code follows Python PEP 8 standards and functions correctly before submitting.
